Characters

GET /characters

Get all the characters from existent campaigns

GET /characters/${campaign}

Get all characters from a campaign

${campaign} = campaign ID that contains the character

GET /characters/${campaign}/${character}

Get a single character from a campaign

${campaign} = campaign ID that contains the character

${character} = The ID of the character you want to get

POST /characters/${campaign}

Add or updates a character. If character already contains an ID, it updates the character with that ID. Otherwise, it generates one and returns the character with it.

BODY: Character to be added/updated

${campaign} = campaign ID that will contain the character

DELETE /characters?campaignId=${obligatory}&characterId=${obligatory}

Deletes a character from a campaign.

${campaignId} = campaign ID that contains the character

${characterId} = The ID of the character you want to delete


Chats

GET /chats

Get all the chats from existent campaigns

GET /chats/${campaign}

Get all chats from a campaign

${campaign} = campaign ID that contains the chat

POST /chats/${campaign}

Add or updates a chat. If chat already contains an ID, it updates the chat with that ID. Otherwise, it generates one and returns the chat with it

BODY: chat to be added/updated

${campaign} = campaign ID that will contain the chat

DELETE /chats?campaignId=${obligatory}&chatId=${obligatory}

Deletes a chat from a campaign.

${campaignId} = campaign ID that contains the chat

${chatId} = The ID of the chat you want to delete

DELETE /chats/clearHistory/${campaign}

Clears the chat history of a campaign.

${campaign} = campaign ID of the chat history

Backup

GET /backup/characters/${campaign}

Get the backup information for the characters of the given campaign's ID

${campaign} = campaign ID that contains the character

POST /backup/characters/${campaign}

Set the backup information for the characters of the given campaign's ID

BODY: The backup information to be set

${campaign} = campaign ID

POST /backup/characters/${campaign}/restore

Restore characters of a campaign using backup data

BODY: The backup data (json)

${campaign} = campaign ID
