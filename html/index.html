Characters

GET /characters

Get all the characters from existent campaigns

GET /characters/${campaign}

Get all characters from a campaign

${campaign} = campaign ID that contains the character

GET /characters/${campaign}/${character}

Get a single character from a campaign

${campaign} = campaign ID that contains the character

${character} = The ID of the character you want to get

POST /characters/${campaign}

Add or updates a character. If character already contains an ID, it updates the character with that ID. Otherwise, it generates one and returns the character with it

BODY: Character to be added/updated. Interface TBD

${campaign} = campaign ID that will contain the character

DELETE /characters?campaignId=${optional}&characterId=${obligatory}

Deletes a character from a campaign. If a campaignId is not introduced, then it deletes all characters with that ID from every campaign

${campaignId} = campaign ID that contains the character

${characterId} = The ID of the character you want to delete
